# Menyimpan dan Mengirim Perubahan ke GitHub

Berikut adalah perintah Git untuk menambahkan perubahan, membuat commit, dan mengunggahnya ke repository remote:

```bash
git add .

git commit -m "belajar-javascript-dasar-hari2-commit2"

git push -u origin main
```

### Penjelasan:
- **`git add .`** → Menambahkan semua perubahan ke staging area.
- **`git commit -m "pesan commit"`** → Menyimpan perubahan dengan pesan deskriptif.
- **`git push -u origin main`** → Mengirim perubahan ke branch `main` di remote repository.

Pastikan untuk mengganti `"belajar-javascript-dasar-hari2-commit2"` dengan pesan yang lebih deskriptif agar mudah dipahami di masa depan. 🚀
